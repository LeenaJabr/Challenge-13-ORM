# Challenge-13-ORM
## The challenge

The challenge is to build the back end for an e-commerce site. Using a working Express.js API and configure it to use Sequelize to interact with a MySQL database.

## Demo





## Usage

- Test API POST, PUT, and DELETE routes in Insomnia Core to successfully create, updated, and delete data in my database.
The following API routes should be created:

- GET `/api/routes` - Insomnia Core for categories, products, or tags.

- POST `/api/routes` - Create/Update/Delete data in database.


## Installation

Download the corresponding files:
- MySQL2
- Sequalize
- dotenv package
- MySQL Workbench

Connect Express.js to MySQL and dotenv - Initiate start on Insomnia.
Use the NPM function "Npm Install" to install, Npm "Run Seed," and "Npm Start" to activate the app.
